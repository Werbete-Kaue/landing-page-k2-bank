Subtítulo profissional: landing page institucional moderna para apresentar uma experiência bancária digital, segura e intuitiva.

## ✨ Funcionalidades

- Design ultra moderno com visual em dark mode, efeitos de vidro e identidade visual futurista.
- Navegação fluida por âncoras entre as seções de Hero, Recursos, Segurança e Sobre.
- Layout totalmente responsivo para desktop, tablet e mobile.
- Seções dedicadas para destacar recursos, segurança, proposta de valor e confiança institucional.
- Estrutura semântica em HTML com estilos modulares em CSS para facilitar manutenção e evolução.
- Animações sutis e microinterações que reforçam a sensação de premium e modernidade.

## 📱 Responsividade

A interface foi pensada para oferecer experiência consistente em diferentes telas. Os principais breakpoints utilizados são:

- 1024px: adaptação para tablets, com reorganização de grids e melhoria da leitura.
- 768px: ajuste para mobile, com empilhamento vertical das seções e menus otimizados para toque.

## 🎨 Guia de Estilos

A identidade visual do projeto utiliza uma paleta escura e sofisticada, com destaque em verde neon para reforçar a sensação de tecnologia, confiança e inovação.

- Fundo principal: #030712
- Destaque principal: #39FF14
- Texto secundário: #5C646E e #9CA3AF
- Elementos de suporte: branco, cinza claro e detalhes em amarelo para reforçar a hierarquia visual.

Esta estrutura facilita a manutenção do projeto e serve como base sólida para futuras evoluções, como integrações com um sistema bancário real ou novas páginas internas.
